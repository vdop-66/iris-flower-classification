Enter sepal and petal measurements to predict the most likely Iris species. The form talks to a lightweight Flask API powered by a StandardScaler + KNN model trained on the classic dataset.

Try a prediction

Numbers can be decimals. Use centimeters for each measurement.

How it works

The backend trains a pipeline when the server starts, then exposes a POST /predict endpoint. Your inputs are scaled and fed into a 3-neighbor classifier.

curl -X POST http://localhost:5000/predict \
  -H "Content-Type: application/json" \
  -d '{"sepal_length": 5.1, "sepal_width": 3.5, "petal_length": 1.4, "petal_width": 0.2}'
